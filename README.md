# PH Fund Tracker

All Philippine Investment Funds — NAVPS Performance Dashboard.

**Live dashboard:** [aiinterruptor.github.io/ph-fund-tracker](https://aiinterruptor.github.io/ph-fund-tracker)

## Features

- 220+ Philippine investment funds tracked (UITFs + Mutual Funds)
- Auto-updated daily at 8:00 AM PHT via GitHub Actions
- Sortable by YTD, 1-year, 3-year, 5-year, 10-year returns
- Filter by category (Equity, Bond, Money Market, Balanced, Feeder)
- Filter by type (UITF vs Mutual Fund)
- Search by fund name or manager
- Top 10 leaderboard
- Mobile responsive

## Data Sources

- [UITF.com.ph](https://uitf.com.ph) — Trust Officers Association of the Philippines
- [PIFA](https://pifa.com.ph) — Philippine Investment Funds Association (via BusinessMirror)
